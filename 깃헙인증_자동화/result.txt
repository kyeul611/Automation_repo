

문제 해결을 위한 로직

1. 입력 받은 방 번호 n의 각 숫자를 세어서 리스트에 저장한다. 
2. 6과 9는 하나의 숫자로 인식되므로 이들의 개수를 합산하고, 2, 5, 6, 9를 제외한 다른 숫자들의 최댓값을 찾아준다. 
3. 만약 최댓값이 6또는9라면, 합산했던 6과 9의 값을 최댓값에 2로 나눈 뒤 올림을 한 값을 사용한다. 
4. 만약 최댓값이 6이나 9가 아니라면, 그 값을 사용한다. 
5. 방 개수를 출력한다. 

코드로 표현하기

```python
nums = [0]*10
n=input()

for i in range(len(n)):
    nums[int(n[i])] += 1

six_and_nine = nums[6]+nums[9]
nums[6] = nums[9] = six_and_nine//2 + six_and_nine%2

max_num = max(nums[:-1])

print(max_num)
```

주의점

1. nums 리스트에 6과 9의 값을 합산할 때, 같은 인덱스에 더해준 후, nums[6]과 nums[9]의 값을 각각 0으로 설정해줘야 한다. 
2. 이 문제는 여러 대의 숫자 판자에 번호가 부여된 상황을 가정하고 있으므로, 같은 숫자를 2번 이상 사용할 수 있다. 
3. 최대한 작은 방 개수를 구해야 하기 때문에, 최대한 6과 9를 활용하여 최댓값을 구해준다.
